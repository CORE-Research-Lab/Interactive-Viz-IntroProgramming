Using a While Loop

Code

searching = True 
counter = 0
wanted_item = 'Bunny'
lst = ['', '', 'Bunny', ''] 
while searching:  
   if (lst[counter] == wanted_item):
       searching = False
   else: 
       counter += 1 
print("Item was found at index", counter)
